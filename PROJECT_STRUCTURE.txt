Database Schema:
================

organisations
├── id (PK)
├── name
└── created_at

users
├── id (PK)
├── organisation_id (FK → organisations)
├── email (UNIQUE)
├── password_hash
├── name
└── created_at

employees
├── id (PK)
├── organisation_id (FK → organisations)
├── first_name
├── last_name
├── email
├── phone
└── created_at

teams
├── id (PK)
├── organisation_id (FK → organisations)
├── name
├── description
└── created_at

employee_teams (Join Table)
├── id (PK)
├── employee_id (FK → employees, CASCADE)
├── team_id (FK → teams, CASCADE)
└── assigned_at

logs
├── id (PK)
├── organisation_id
├── user_id
├── action
├── meta (JSONB)
└── timestamp


API Endpoints:
==============

Authentication:
- POST   /api/auth/register          # Create organisation & admin
- POST   /api/auth/login             # User login
- POST   /api/auth/logout            # User logout (protected)
- GET    /api/auth/me                # Get current user (protected)

Employees:
- GET    /api/employees              # List all employees (protected)
- GET    /api/employees/:id          # Get single employee (protected)
- POST   /api/employees              # Create employee (protected)
- PUT    /api/employees/:id          # Update employee (protected)
- DELETE /api/employees/:id          # Delete employee (protected)

Teams:
- GET    /api/teams                  # List all teams (protected)
- GET    /api/teams/:id              # Get single team (protected)
- POST   /api/teams                  # Create team (protected)
- PUT    /api/teams/:id              # Update team (protected)
- DELETE /api/teams/:id              # Delete team (protected)

Team Assignments:
- POST   /api/teams/:id/assign       # Assign employees to team (protected)
- DELETE /api/teams/:id/unassign/:empId  # Remove employee from team (protected)

Logs:
- GET    /api/teams/logs/all         # Get activity logs (protected)

Quick Start:
============

1. Install PostgreSQL and create database 'hrms_db'

2. Backend Setup:
   cd backend
   npm install
   # Update .env with your DB credentials
   npm run seed
   npm run dev

3. Frontend Setup (new terminal):
   cd frontend
   npm install
   npm run dev

4. Open browser: http://localhost:3000

5. Login with demo credentials:
   Email: admin@demo.com
